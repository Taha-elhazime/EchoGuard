title: EchoGuard
author: Taha kobbi El Hazime
description: Echoguard is a security system that is meant to be more effiecent than convencional cameras. The main part is a camera turret which is ceiling mounted and uses microphones in order to detect suspecious activity and lock on to it. It also uses an ultrasonic sensor that sticks to the wall and uses bluetooth to detect anyone who activates it.
